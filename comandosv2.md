# Ramas

Crear y cambiarte a una rama
```Shell
git checkout -b nombre_rama
o
git switch -c nombre_rama
```
Eliminar rama
```Shell
git branch -D nombre_rama
```

Crear y cambiarte a una rama
```Shell
git checkout -b nombre_rama
```

Combinar una rama con main
```Shell
git checkout main
git merge nombre_rama
```
